# 🦕 体調モニター - 恐竜育成アプリ

毎日の体調を記録して、可愛い恐竜を育てる健康管理PWAアプリです。

## ✨ 特徴

- **シンプルな記録**: 3ステップで体調を記録
- **恐竜育成**: 連続記録で恐竜が成長
- **アンロックシステム**: 頑張ると新しい恐竜が解放
- **オフライン対応**: インターネットがなくても使用可能
- **データ管理**: バックアップ・復元機能付き

## 📱 Android での使い方

### 1. インストール

1. AndroidのChromeブラウザで次のURLを開く
   ```
   https://[あなたのGitHubユーザー名].github.io/[リポジトリ名]/
   ```

2. 画面右上のメニュー（⋮）をタップ

3. 「ホーム画面に追加」を選択

4. アプリ名を確認して「追加」

5. ホーム画面にアイコンが追加されます

### 2. アプリとして使用

- ホーム画面のアイコンからアプリを起動
- ネイティブアプリのように全画面表示
- オフラインでも動作
- データは端末内に安全に保存

## 🎮 使い方

### 体調を記録

1. **気分を選ぶ**: 「良い」「普通」「悪い」から選択
2. **数値を選ぶ**: 具体的な体調値（1〜10）を選択
3. **メモを追加**: 任意で一言メモを入力

### 恐竜の成長

- **1日目**: タマゴが孵化
- **8日目**: 子供に成長
- **22日目**: 大人に成長
- **30日目**: サイクル完了、新しい恐竜を選択

### アンロック

連続記録を続けることで新しい恐竜が解放されます：
- **0日**: ティラノサウルス、トリケラトプス、プテラノドン（初期）
- **25日**: スティラコサウルス
- **35日**: ステゴサウルス
- さらに多くの恐竜...

## 🛠️ 技術スタック

- **HTML/CSS/JavaScript**: Vanilla JS（フレームワーク不使用）
- **PWA**: Service Worker + manifest.json
- **ストレージ**: LocalStorage
- **アイコン**: PWA対応の複数サイズアイコン

## 📂 ファイル構成

```
for_mobile/
├── index.html              ← メインアプリ
├── manifest.json           ← PWA設定
├── sw.js                   ← Service Worker
├── dinosaur-species.js     ← 恐竜データ
├── .gitignore              ← Git設定
├── README.md               ← このファイル
├── js/
│   ├── date-utils.js       ← 日付処理
│   ├── storage-utils.js    ← データ管理
│   ├── data-manager.js     ← 成長システム
│   └── unlock-system.js    ← アンロック機能
└── assets/
    ├── characters/         ← 恐竜画像（24種）
    └── icons/              ← PWAアイコン
```

## 🚀 GitHub Pages でのホスティング

### 1. リポジトリ作成

```bash
git init
git add .
git commit -m "Initial commit: Health Monitor PWA"
git branch -M main
git remote add origin https://github.com/[username]/[repository].git
git push -u origin main
```

### 2. GitHub Pages 有効化

1. GitHubリポジトリの「Settings」タブ
2. 左メニューの「Pages」
3. Source: `main` branch / `root`
4. 「Save」をクリック

### 3. アクセス

数分後、以下のURLでアクセス可能：
```
https://[username].github.io/[repository]/
```

## 💾 データ管理

### バックアップ

1. アプリ内の⚙️（設定）ボタンをタップ
2. 「データのバックアップ」セクション
3. 「データをコピー」ボタンでクリップボードにコピー
4. テキストファイルなどに保存

### 復元

1. 設定画面の「データの復元」セクション
2. バックアップデータを貼り付け
3. 「データを復元」ボタンをタップ

## 🔔 通知機能

毎朝7:30に記録リマインダー通知を送信（オプション）：
- 設定画面で「通知設定をONにする」
- ブラウザの通知許可が必要
