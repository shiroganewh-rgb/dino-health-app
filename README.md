# 体調モニタリングアプリ (PWA版) 導入ガイド

このフォルダには、Androidスマートフォンで動作するアプリのファイル一式が含まれています。

## 📲 インストール手順 (Android)

1. **ファイルの準備**
   このフォルダ (`for_mobile`) の中身を全て、あなたの **Google Drive** などのクラウドストレージにアップロードしてください。
   （特定のフォルダを作ってそこに入れると管理しやすいです）

2. **アプリを開く**
   Androidスマートフォンの **Chromeブラウザ** で、アップロードした `app-working.html` を開きます。
   ※ドライブアプリから直接開くとChromeで開けない場合があるため、Chromeのアドレスバー等を利用するか、「アプリで開く」→「Chrome」を選択してください。
   ※ローカルのHTMLファイルとして開く必要があります。

3. **ホーム画面に追加 (インストール)**
   画面が開いたら、ブラウザのメニュー（右上の︙）を開き、
   **「ホーム画面に追加」** または **「アプリをインストール」** をタップします。

4. **完了！**
   ホーム画面に恐竜のアイコンが追加されます。
   これ以降は、そのアイコンをタップするだけで、インターネットがなくてもアプリを使用できます。

## 🔄 更新手順

アプリの機能がアップデートされた場合は、以下の手順で更新します。

1. 新しいファイルをGoogle Driveに上書きアップロードします。
2. アプリを開き、**ページを再読み込み（リロード）** します。
3. 自動的に新しいバージョン（新しい恐竜など）が反映されます。
