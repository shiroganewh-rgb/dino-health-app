# 体調モニタリングアプリ - 恐竜育成日記 🦖

毎日の体調を記録して、かわいい恐竜たちを育てるアプリです。
あなたの健康管理が、恐竜たちの成長につながります！

## 🌟 アプリでできること

### 1. 毎日の体調記録
- その日の体調を **1（悪い）〜10（絶好調）** の数字で選ぶだけ。
- **AM 4:00** に日付が変わるので、深夜に記録しても前日分として扱われます。
- 簡単なメモも残せます。

### 2. 恐竜の育成
- 記録をつけると、恐竜に経験値が入ります。
- **連続で記録**すると、より早く成長します！
- タマゴ → 赤ちゃん → 子供 → 大人 と4段階に進化します。

### 3. パートナー交代（月次リセット）
- **毎月1日** になると、新しいパートナーを選ぶ画面になります。
- 毎月違う種類の恐竜を育てて、コンプリートを目指しましょう！
- 育てた記録は「図鑑」に残ります。

### 4. 恐竜図鑑
- 今までに育てた恐竜の姿をいつでも振り返れます。
- 下のアイコンをタップすると、タマゴや赤ちゃん時代の姿も見られます。

---

## ⚠️ 重要：データについて

このアプリは会員登録などが不要で、あなたのスマホの中にだけデータを保存します。
そのため、以下の点にご注意ください。

> **【絶対にアプリを削除しないでください】**
> ホーム画面からアイコンを削除（アンインストール）すると、**今までの記録や育てた恐竜のデータが全て消えてしまいます**。
> データの復旧はできません。

---

## 📲 使い方ガイド

### 毎日やること
1. アプリを開く。
2. 今日の体調（1〜10）を選ぶ。
3. 一言メモを入力（スキップOK）。
4. 「記録する」ボタンを押す。
   → 恐竜が成長するかも！？

### カレンダー機能
- 画面左上の「カレンダー」アイコンから、過去の記録を確認できます。
- 過去の記録を修正・削除することも可能です。

### アイコンが変わらない時
- アプリのアイコンデザインが新しくなったのに変わらない場合、アプリを消さずに**「画面を下に引っ張って更新」**したり、何度か起動し直して数日待ってみてください。自動的に更新されます。

---

## 🛠️ インストール方法 (Android)

まだインストールしていない場合は、以下の手順でスマホに入れてください。

1. **準備:** このフォルダの中身を全て Google Drive にアップロードします。
2. **開く:** スマホの Chrome ブラウザで `index.html` または発行されたURLを開きます。
3. **追加:** ブラウザのメニュー（︙）から **「ホーム画面に追加」** または **「アプリをインストール」** を選びます。

これで、インターネットがなくても動く自分だけのアプリが完成します！
